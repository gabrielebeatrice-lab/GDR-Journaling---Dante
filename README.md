# GDR-Journaling---Dante
Materiali, esempi e linee guida per introdurre il GDR Journaling nella didattica. Un approccio narrativo basato sulla scrittura in prima persona, utile per sviluppare creatività, competenze metacognitive e capacità di analisi testuale. La repo contiene modelli, schede, prompt e attività pronte per l’uso in classe.
